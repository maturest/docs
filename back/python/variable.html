在了解变量之前，我们先宏观的理解一下计算机硬件系统的五大部件：运算器，控制器，存储器，输入设备，输出设备。 程序是指令的集合，运算器和控制器放在一起就是我们通常说的中央处理器，它的功能是执行各种运算和控制指令以及处理计算机软件中的数据。 计算机中的变量是实际存在的数据，是存储器中存储数据的一块内存空间。

# 变量类型

常见类型有：整型，浮点型，字符串，布尔型，列表，元组，字典等等。

# 变量与变量名

变量存在于我们电脑的内存里，每个变量被定义后存入一个内存块。

变量名的规则：

1、组成：数字字母下划线组成

2、长度：任意长度，但不宜太长

3、要求：开头必须是字母或者下划线

4、注意：区分大小写，有意义

5、变量名推荐使用下划线的方式

6、不能使用关键字

# 关键字

变量名用于给变量赋值使用，而关键字用于业务逻辑处理

关键字的分类：

  • 强关键字（完全不能用来定义变量名）

  • 弱关键字（可以用来变量名，但会影响功能）

常见的关键字：

  • True
  • False
  • def
  • if
  • elif
  • else
  • try
  • is
  • not
  • or
  • pass
  • raise
  • in
  • while
  • with
  • yield
  • import

  • int()：将一个数值或者字符串转换成整数

  • float()：将一个字符串转化成浮点数

  • str()：将指定的对象转换成字符串

  • chr()：将整数转换成该编码对应的字符串

  • ord()：将字符串转换成对应的编码

# 练习

将华氏温度转换为摄氏温度，


'''
   参考 c = (f - 32) / 1.8
'''

f = float(input('请输入华氏温度'))
c = (f - 32) / 1.8
print('%.1f华氏度 = %.1f摄氏度' % (f,c))

输入圆的半径计算周长和面积


'''
   计算周长和面积
'''
r = float(input('请输入半径：'))

perimeter = 2 * 3.14 * r
area = 3.14 * r**2

print('周长%.2f' % perimeter)
print('面积%.2f' % area)

输入年份判断是否是闰年


'''
   判断年份是否是闰年
'''
year = int(input('请输入年份：'))

is_leep = year % 4 == 0 and year % 100 != 0 or year % 400 == 0 

print(is_leep)
