# 集合的定义

集合是一个无序的不重复元素序列，常用来对两个列表进行交并差的处理。和列表一样，支持所有数据类型。

# 集合与列表的区别

功能 列表 集合
顺序 有序 无序
内容 可重复 不可重复
功能 用于数据的使用 用于数据的交集并集差集的获取
索引 有索引 无索引
符号 [] {}

# 集合的创建

通过 set 函数来创建集合，不能使用 {} 来创建空集合。

a = set()
b = set([1,2,3]) # 把列表中的成员放到集合里
c = {1,2,3}
d = {} # 字典来着

# 集合的增删改

  • add
set.add(item)
  • update
# 参数类型 集合，列表，元组，字符串
set.update(iterable)
  • remove 将集合中的某个元素进行删除
set.remove(item)
  • clear 清空集合中的所有元素
set.clear()
  • del
del set

# 两个集合的操作

  • 差集
a_set.difference(b_set)
  • 交集
a_set.intersection(b_set)
  • 并集
a_set.union(b_set)
  • 符号操作
符号 函数
& set1.intersection(set2)求set1和set2的交集
| set1.union(set2) 求set1和set2的并集
- set1.difference(set2) 求set1和set2的差集
^ set1.symmetric_difference(set2) 求set1和set2的交叉补集

# 判断两个集合中是否有相同的元素

a_set.isdisjoint(b_set)
