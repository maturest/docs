# 字典的常用方法

# 获取字典的所有 keys

keys 返回一个 key 集合的伪列表，可以使用类型转换函数 list 将其转换为真列表。

project = {'id':1,'project_name':'iphone','price':8000}

project_keys = project.keys()

# <class 'dict_keys'>
type(project_keys) 

list(project_keys)

# 字典 key 的获取

  • [] 方式获取
project = {'id':1,'project_name':'iphone','price':8000}

project_name = project['project_name']
  • get 用法
project = {'id':1,'project_name':'iphone','price':8000}

project_name = project.get('project_name','default_value')

# 字典删除

  • clear 清空当前字典中的所有数据
dict.clear()
  • pop 删除字典中指定的key，并将其结果返回
dict.pop(key)
  • del
del dict['key']

# 字典的复制

  • copy 将当前字典复制成一个新的字典，返回一模一样的内存地址不同的字典
new_dict = old_dict.copy()

# 字典的成员运算符

# 字典 popitem

  • popitem 删除当前字典里末尾一组键值对并将其返回
dict = {'name':'shuai','age':20}


dict.popitem() # ('age',20)
