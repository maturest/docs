# 字符串的格式化操作

  • 字符串格式化操作
'%d * %d = %d' % (a,b,a*b)

'{0} * {1} = {2}' .format(a,b,a*b)

f'{a} * {b} = {a * b}'
  • capitalize 字符串首字母大写
name = 'hello Kitty'
new_name = name.capitalize()
  • casefold 与 lower 字符串转小写
string.casefold()
string.lower()
  • upper 字符串大写
string.upper()
  • swapcase 字符串的字母大小写转换（大写转小写，小写转大写）
string.swapcase()
  • zfill 为字符串定义长度，如果不足则填零
string.zfill(width)
  • count 统计指定字符出现的次数
string.count('substr')
  • startswith 开始的位是否是指定成员
string.startswith()
  • endswith 结束位置是否指定成员
string.endswith()
  • find 查找指定的元素，返回一个整数,找不到就返回 -1
string.find(item)
  • index 查找指定的元素，返回一个整数，找不到就报错
string.index(item)
  • strip 去掉字符串两遍的指定元素，默认去除两边的空格
string.strip(item)
string.lstrip(item)
string.rstrip(item)
  • replace 替换字符串，并且可以指定替换数量，默认替换全部
string.replace(old,new)
  • 字符串中返回 bool 类型的函数集合
isspace 判断字符串是否是一个由空格组成的字符串
istitle 判断字符串是否是一个标题类型
isupper 判断按字符串中的字母都是大写
islower 判断按字符串中的字母都是小写
