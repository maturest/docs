# python 中的数据类型

数字类型、字符串类型、布尔类型、空类型、列表类型、元组类型、字典类型

# 数字类型及其应用

整型，浮点型

num1 = int(100)
num2 = 100

num3 = float(3.14)
num4 = 3.14

type 函数返回变量的类型

count = 100
print(type(count))

# 字符串类型及其应用

str = str('abc')
str1 = 'abc'

字符串不可改变

内置函数 id 返回变量的内存地址

内置函数 len 返回字符串的长度，无法返回数字类型的长度，因为数字类型没有长度。

内置成员运算符 in 的使用

max 函数返回数据中最大的成员，中文字符 > 字母 > 数字 > 英文符号

min 函数返回数据中最小的成员

字符串的累加 字符串拼接用 +

# 布尔类型与空类型

0 => false 0.0 => false '' => false None => false [] => false () => false {} => false

# 列表类型

列表就是队列，是各种数据类型的集合，也是一种数据结构，是一种有序，且内容可重复的集合类型。虽然是一个无限长度的数据结构，但避免出现长度过长。

怎么定义：

name1 = list(['张三', '李四', '王二'])
name2 = ['张三', '李四', '王二']

max,min在列表中使用的话，列表中的元素类型要统一。

# 元组类型

元组与列表一样，都是一种可以存储多种数据结构的队列

元组也是一个有序的，且元素可以重复的集合

定义：

names = tuple(('张三','李四'))
names = ('张三','李四')
names = ('张三',) # 一个成员的时候，要带逗号，当不加逗号的时候，该变量的类型就是该成员的类型。

列表与元组的区别：

  • 元组比列表占用资源更小
  • 列表是可变的，元组是不可变的

# 字典

定义：

a = dict()
a = {}
person = {'name': '张三', 'age': 23}

key 支持字符串，数字，元组，但列表是不支持的

注意以上列表、元组，字典的书写规范。

# 赋值与比较运算符

赋值运算符

  • =
  • +=
  • -=
  • *=
  • /=
  • %=
  • **=
  • //=

字符串，列表，元组是可以使用乘法的，但字典是不可以的

比较运算符

  • ==
  • !=
  • <

  • =

  • <=
  • <>
  • is 判断两个对象存储单元是否相同
  • is not 判断两个对象存储单元是否不同 存储单元就是我们之前讲的 内存块

有趣的现象： 在终端使用 python 解释器的时候，会事先将 0-256 之间的数字分配到内存当中，当 a,b = 200,200,此时 a is b 返回 true。当 a,b = 257,257 的时候，a is b 返回 false。

在脚本中，代码是从上至下执行的，a,b = 200,200，此时 a is b 返回的结果是 True。是同一块内存地址。
