# 列表元组的操作

  • len 在列表元组中的使用
names = ['张三','李四']
length = len(names)
  • 列表之间的累加与乘法
names = ['张三','李四']
new_names = names + names
print(new_names)

new_names = names * 2
print(new_names)
  • in 与 not in 在列表或元组中的使用
names = ['张三','李四']
name = '张三'
print(name in names)
print(name not in names)
  • append 在尾部添加元素
list.append(new_item)
  • insert 将元素添加到指定位置
fruits = ['apple','banana']
fruits.inster(1,'orange')
  • count 想要查询个数的元素
list.count(item)
  • remove 删除列表中的某个元素,有多个元素的话，删除第一个
list.remove(item)
  • del 把变量完全删除
drinks = ['雪碧','可乐','芬达']
del drinks
  • reverse 对当前的列表顺序进行反转
list.reverse()
  • sort 对当前列表按照制定规则进行排序
list.sort(cmp,key,reverse)
  • clear 将当前列表进行清空
list.clear()
  • copy 将当前列表复制一份新的列表
list.copy()

二次赋值的变量与原始变量享有相同的内存空间

copy函数创建的新列表与原始类表不是一个内存空间，不同享数据变更。

copy 属于浅拷贝，通俗的说，我们有一个列表a,列表里的元素还是列表，当我们拷贝出新列表b后，无论是a还是b的内部的列表的数据发生变化后，相互之间都会受到影响。

深拷贝对第一层数据进行了 copy,对深层的数据也进行了 copy,原始变量和新变量完全全不共享数据

  • extend 将其他列表或元组中的元素导入到当前列表中
list.extend(iterable)
