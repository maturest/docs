# vscode 编辑器的使用小技巧

# 如何使用VScode创建blade文件

1、安装插件 Laravel Create View

2、快捷键 ctrl + shift + p 输入命令 laravel create view

3、输入目标文件名称，目录使用 . 间隔，例如 orders.index

# 如何快速跳转至指定文件

可以跳转至控制器，配置文件，视图文件等等。

1、安装插件 Laravel Goto ，并使用 alt + ; 快捷键实现文件跳转

2、同样可以安装别的 laravel goto xxx 插件

# 如何预览 markdown 文档

1、安装插件 markdown preview enhanced

2、鼠标右键选择预览即可

# 如何快速使用 blade

1、安装插件 laravel blade snippets

2、安装插件 bootstrap5 snippets

两者搭配，快速开发
