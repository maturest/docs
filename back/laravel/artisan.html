# 常用的 Artisan 命令

命令 解释
php artisan key:generate 生成 App Key
php artisan migrate 执行迁移
php artisan migrate:rollback 回滚迁移
php artisan migrate:refresh 重置数据库
php artisan db:seed 填充数据库
php artisan tinker 进入 tinker 环境
php artisan route:list 查看路由列表
php artisan event:generate 生成缺省的事件或者事件监听器
php artisan make:command 生成 artisan 命令
php artisan make:controller 生成控制器
php artisan make:event 生成事件类
php artisan make:exception 生成异常类
php artisan make:factory 生成模型工厂类
php artisan make:job 生成任务类
php artisan make:listener 生成事件监听器
php artisan make:mail 生成邮件类
php artisan make:middleware 创造中间件
php artisan make:migration 生成迁移文件
php artisan make:model 生成模型类
php artisan make:notification 生成通知类
php artisan make:observer 生成模型观察器
php artisan make:policy 生成授权策略类
php artisan make:provider 生成服务提供者
php artisan make:request 生成请求类
php artisan make:resource 生成资源
php artisan make:rule 生成验证规则
php artisan make:scope 生成局域类
php artisan make:seeder 生成种子类
php artisan make:test 生成测试类
php artisan notifications:table 生成数据库消息通知频道
