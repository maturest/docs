# PHP Redis 扩展

在 php 中使用 redis 前，需要确保已经安装了 redis 服务和 redis 驱动。

# 安装驱动

安装 phpredis 驱动下载地址为:https://github.com/phpredis/phpredis/releases (opens new window)。

# 下载源码
wget https://github.com/phpredis/phpredis/archive/3.1.4.tar.gz

# 解压
tar zxvf 3.1.4.tar.gz   

# 进入 phpredis 目录               
cd phpredis-3.1.4          

# php安装后的路径            
/usr/local/php/bin/phpize   

# 配置
./configure --with-php-config=/usr/local/php/bin/php-config

# 安装
make && make install

安装成功后将扩展添加到 php.ini 配置文件当中，之后再重启 php-fpm，查看是否成功安装 phpredis 扩展。

extension_dir = "/usr/local/php/lib/php/extensions/no-debug-zts-20090626"

extension=redis.so

在 PHP 项目中，怎么使用呢，可以参考官方文档进行操作：https://phpredis.github.io/phpredis/Redis.html (opens new window)
